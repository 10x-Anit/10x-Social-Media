How It Works

From clone to first post in 10 minutes. Claude handles everything.

01

📦 Clone & Setup

Clone the repo, open Claude Code, type /setup. Claude installs Docker, Node.js, creates configs, starts services — all automatically.

02

🔗 Connect Accounts

Click a Composio link to authorize each platform. No developer apps needed. Support for 13 platforms including LinkedIn, Twitter, Instagram.

03

✍️ Create & Publish

Say "post about our launch on LinkedIn" — Claude drafts in your voice, shows preview, publishes on approval. Schedule posts for the future.

04

📊 Track & Analyze

Per-post metrics tracked over time: likes, comments, impressions, reach, engagement rate. Weekly summaries and content type analysis.

05

🔄 Repurpose

Turn a LinkedIn post into a Twitter thread, Instagram caption, or TikTok script. Claude adapts format, tone, and length per platform.

06

📱 Access Anywhere

Terminal, mobile app, Telegram, Slack, Discord, WhatsApp, or browser dashboard. All share the same backend — everything stays in sync.

Commands

10 slash commands. Each one is self-contained with pre-flight checks, output templates, and error handling.

/setup

Full automated setup. Installs dependencies, creates configs, connects accounts, personalizes voice. Start here.

/post

Create & publish. Draft in your voice, preview, approve, publish to any platform. Multi-account support.

/draft

Draft only. Write content without publishing. No API calls needed. Hand off to /post or /schedule.

/schedule

Future delivery. Pick optimal times from your calendar. Temporal Cloud ensures reliable delivery.

/analytics

Post performance. Likes, comments, impressions, reach, engagement rate. Compare posts, weekly summaries.

/track-analytics

Time-series tracking. Capture metrics at hour 0, day 1, day 7, day 30. Growth curves and trends.

/browse-social

Browser automation. Works on any page. Accessibility tree parsing, no hardcoded selectors.

/repurpose

Cross-platform adapt. LinkedIn → Twitter thread. Blog → carousel. Any format to any format.

/audit

Profile check. Scores each section of your profile. A/B/C/D grade with improvement recommendations.

/index-check

System integrity. Validates all cross-references, file paths, and index consistency.
