You are repurposing existing content for different platforms.

## Steps

1. **Get** the source content (paste, file, or URL)
2. **Ask** target platform(s) and format(s)
3. **Read** skills/social-voice.md and skills/post-formats.md
4. **Read** the target template from templates/
5. **Apply** cross-platform adaptation rules from post-formats.md
6. **Draft** the adapted version(s)
7. **Show** side-by-side: original vs. adapted
8. **Offer** to publish via /post or save via /draft

## Common Transformations
- LinkedIn post → Twitter thread
- LinkedIn post → Instagram caption
- Blog post → LinkedIn carousel outline
- Twitter thread → LinkedIn narrative
- Any → TikTok script
- Any → YouTube outline
