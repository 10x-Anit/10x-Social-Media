## LinkedIn Formats

### Text Post (most common)
```
[Hook — 1 compelling line that stops the scroll]

[2-3 short paragraphs, one idea each]
[Use line breaks between thoughts]

[Insight or lesson learned]

[CTA — question or next step]

#hashtag1 #hashtag2 #hashtag3
```
Max: ~3,000 characters. Sweet spot: 800-1,200.

### Carousel Post
Each slide:
- Slide 1: Title slide (bold claim or question)
- Slides 2-8: One point per slide (large text, minimal words)
- Last slide: CTA + profile mention
Format: PDF or multi-image upload.

### Poll
- Question: clear, binary or scaled
- 2-4 options (keep simple)
- Add context in post body above the poll

### Article (long-form)
- Title: clear benefit + keyword
- 800-2,000 words
- Subheadings every 200-300 words
- End with discussion question

## Twitter/X Formats

### Single Tweet
Max 280 chars. Front-load the point.

### Thread
- Tweet 1: Hook + "Thread 🧵"
- Tweets 2-8: One point each
- Last tweet: Summary + CTA + "Follow for more"

## Facebook Formats
- Text: up to 63,206 chars (but 100-250 words optimal)
- Link posts: short intro + URL
- Photo/Video: caption under 125 chars for mobile

## Instagram Formats
- Caption: up to 2,200 chars
- First line visible before "more" — make it count
- Hashtags: 5-15, at end or in first comment

## TikTok Formats
- Video description: up to 2,200 chars
- Hashtags: 3-5 trending + niche
- Hook in first 3 seconds of video

## YouTube Formats
- Title: under 60 chars, keyword first
- Description: 2-3 sentence summary, then timestamps, then links
- Tags: 5-10 relevant keywords

## Cross-Platform Adaptation Rules
When repurposing content:
1. LinkedIn post → Twitter thread: break into atomic points
2. LinkedIn post → Instagram: add visual, move text to caption
3. Twitter thread → LinkedIn: combine into flowing narrative
4. Any → TikTok: extract the hook, write a 30-60s script
5. Any → YouTube: expand into 5-10 min outline with intro/outro
