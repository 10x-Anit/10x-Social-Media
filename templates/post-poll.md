# Poll Post Template
<!-- [F:TMPL.03] Poll post structure -->

## Structure

```
{{CONTEXT_PARAGRAPH — why this question matters}}

{{POLL_QUESTION}}

Option 1: {{OPTION_A}}
Option 2: {{OPTION_B}}
Option 3: {{OPTION_C}} (optional)
Option 4: {{OPTION_D}} (optional)

{{FOLLOW_UP — "I'll share my take in the comments"}}
```

## Rules
- Question should be clear and answerable
- Options should be distinct (no overlapping)
- 2-4 options (fewer = higher engagement)
- Add "Other (comment below)" as last option for discussion
- Post your own answer as first comment
