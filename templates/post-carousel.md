# Carousel Post Template
<!-- [F:TMPL.02] Carousel/document post structure -->

## Slide Structure

```
Slide 1 (Title):
  {{BOLD_CLAIM_OR_QUESTION}}
  {{SUBTITLE — why they should swipe}}

Slides 2-8 (Content):
  {{POINT_NUMBER}}/{{TOTAL}}
  {{MAIN_POINT — large text, under 15 words}}
  {{SUPPORTING_DETAIL — 1-2 sentences}}

Last Slide (CTA):
  {{SUMMARY_OF_KEY_POINTS}}
  {{FOLLOW_CTA}}
  {{PROFILE_MENTION}}
```

## Companion Post Text

```
{{HOOK_ABOUT_THE_CAROUSEL}}

Here's what's inside:
{{BULLET_LIST_OF_SLIDE_TOPICS}}

Save this for later.

{{HASHTAGS}}
```

## Rules
- 6-10 slides optimal
- One point per slide
- Large readable text (works on mobile)
- Consistent visual style across slides
- Upload as PDF or multi-image
